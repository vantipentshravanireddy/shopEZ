# ShopEZ

ShopEZ is a full-stack E-Commerce web application developed using the MERN Stack (MongoDB, Express.js, React.js, Node.js).

## Features

* User Registration and Login
* Product Listing and Search
* Shopping Cart Management
* Order Placement
* Order History Tracking
* Admin Dashboard
* Product Management
* Revenue Analytics

## Technology Stack

### Frontend

* React.js
* Vite
* React Router DOM
* Axios
* CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB

## Project Architecture

Frontend (React.js) → Backend (Node.js + Express.js) → MongoDB Database

## Modules

### User Module

* Registration
* Login
* Profile Management

### Product Module

* Product Listing
* Product Search
* Category Filtering

### Cart Module

* Add to Cart
* Remove from Cart
* Quantity Management

### Order Module

* Place Order
* View Order History
* Order Status Tracking

### Admin Module

* Dashboard Statistics
* Product Management
* Order Monitoring

## Installation

### Backend

```bash
cd server
npm install
npm start
```

### Frontend

```bash
cd client
npm install
npm run dev
```
